```?help Shows commands
?ping Play ping-pong
?status View status of the bot
?question The bot will ask random question
?timer <minutes> Make the bot ping you in x amount of minutes
------------------------------***Staff only***------------------------------
?kick <user> Kicks user 
?ban <user> Bans user 
?unban <user> Unbans user 
?announce <channel> <message> Make announcements 
?purge <number> Purges messages 
?kill Terminate the bot 
?reboot Reboots the bot 
?warn <user> <reason> Warns user 
?clearwarn <user> Deletes warning(s) for the specified user 
?listwarn <user> List user warnings 
?mute <user> <reason> Mutes specified user 
?unmute <user> Unmutes specified user 
?addrole <user> <role> Adds user to a role
?delrole <user> <role> Deletes user from a role 
?createrole <role> Creates a role
?jail <user> Jails user```
